# ADG Technologies - Full Stack Application

This project is a full-stack web application for ADG Technologies, featuring a React frontend and a Node.js backend with authentication.

## Structure

- `/client`: React frontend (Vite)
- `/server`: Node.js/Express backend
- `.env`: Environment variables (Mongo URI, JWT Secret)

## Getting Started

### Backend
1. `cd server`
2. `npm install`
3. `node server.js`

### Frontend
1. `cd client`
2. `npm install`
3. `npm run dev`

## Features
- User Registration & Login
- JWT Authentication
- Protected Dashboard
- Tailwind CSS Styling
